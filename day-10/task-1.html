About Me

  1. Read the input file containing lines of code with various brackets.
  2. For each line, check for syntax errors by ensuring that all opening brackets have corresponding closing brackets in the correct order.
  3. If a line contains a syntax error, identify the first incorrect closing bracket and calculate a score based on predefined values for each type of bracket.
  4. Sum the scores of all syntax errors to get the total syntax error score.
  5. For incomplete lines (those without syntax errors but with unclosed brackets), determine the sequence of closing brackets needed to complete the line.
  6. Calculate a completion score for each incomplete line based on the required closing brackets and their respective values.
  7. Sort the completion scores and find the median score to determine the final result for incomplete lines.
  8. Output both the total syntax error score and the median completion score as the final results.
